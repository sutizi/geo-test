# Challenge Geopagos: torneo de tenis

## Endpoints:

- /player

Propiedades de los players:
 - Name
 - Gender (F o M)
 - Skill: conjunto de habilidades del jugador (force, velocity, reaction_time)

- /tournament

Propiedades de los tournament:
 - Name
 - Date
 - Winner (player ganador del torneo)
